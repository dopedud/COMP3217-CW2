Steps to re-run training on machine learning models:
1. With Google Notebook project opened, upload corresponding training and test files (e.g. for binary classification models, use TrainingDataBinary.csv and TestingDataBinary.csv)
2. Choose 'Run All' option that is hidden under 'Runtime' tab in the taskbar located at the top.
3. When run, cell 2 will ask for files to be uploaded. Cancel this operation.
